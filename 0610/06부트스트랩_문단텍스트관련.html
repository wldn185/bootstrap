06부트스트랩_문단텍스트관련

부트스트랩에서는 기본 글자 크기 16px, 행높이(줄간격)은 1.428값이 기본이다.

위 내용은 문단태그(p)요소에 자동으로 적용되며, 하단여백(margin)은 16px이 기본값이다.

화면이 100%비율이고 전체화면일 경우 아래 값이 나오며 가로크기가 변하면 유동적으로 값이 변경됨.

제목1 - 40px

제목2 - 32px

제목3 - 28px

제목4 - 24px

제목5 - 20px
제목6 - 16px

제목요소의 마진값은 기본이 8px로 설정(아래)

small - 글자를 작게 설정함.(14px) mark - 글자에 하이라이트 효과를 적용함.(16px) abbr - 약어나 줄임말을 사용하고자 할 때 사용.(16px)

blockquote - 다른사람의 말이나 글을 대신하여 작성하는 글(16px) class="blockquote-footer"일 경우(14px, 색상 : #6c757d)
애플 ceo 스티브잡스

우리의 실패는 경험으로부터 강해졌다.

blockquote는 클래스명 blockquote-footer를 적용하면 글자크기, 색상, 줄이 생김.

dl, dt, dd태그 적용시 크기 16px로 동일함.

dt - 용어제목
dd - 용어설명
code : 문서안에 태그 이름을 표시하기 위한 태그로서 색상이 '자주색'으로 변경됨. let gnb = document.getElementById('gnb');

text-muted : 텍스트의 중요도에 따라서 문자의 색상을 다르게 표시할 수 있다.(회색)

text-primary : 텍스트의 중요도에 따라서 문자의 색상을 다르게 표시할 수 있다.(파랑색)

text-success : 텍스트의 중요도에 따라서 문자의 색상을 다르게 표시할 수 있다.(초록색)

text-info : 텍스트의 중요도에 따라서 문자의 색상을 다르게 표시할 수 있다.(하늘색)

text-warning : 텍스트의 중요도에 따라서 문자의 색상을 다르게 표시할 수 있다.(주황색)

text-danger : 텍스트의 중요도에 따라서 문자의 색상을 다르게 표시할 수 있다.(빨강색)

상황별로 문자의 배경색을 적용하려면 class="bg_primary", class="bg_success", class="bg-dark", class="bg_warning", class="bg_danger"을 p요소안에 작성한다.

문단관련서식

class=lead : 단락을 눈에 띄게 하기 (20px)

class=small : 상위문자 크기의 85%=14px로 문자크기를 작게 설정함.

class=text-start : 문단정렬(왼쪽정렬)

class=text-end : 문단정렬(오른쪽정렬)

class=text-center : 문단정렬(가운데정렬)

class=text-justify : 문단정렬(양쪽배분정렬)

class=text-nowrap : 문단끝에서 글자가 다음줄로 넘어가지 않기

class=text-uppercase : 영문자를 무조건 대문자로 출력

class=text-lowercase : 영문자를 무조건 소문자로 출력

class=text-capitalize : 영문자의 첫글자만 대문자로 출력
